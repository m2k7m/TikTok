<h1>TikTok Account Info</h1>
<h2 style="border-bottom: none;">Useful script for extracting TikTok account information</h2>

<h2>Requirements:</h2>

Python 3.12.x or higher

```bash
pip install requests bs4 countryflag
```

Make sure to download them all.

<h2>How to use:</h2>
<ul>
    <li>Install requirements</li>
    <li>Download as ZIP</li>
    <li>Run main.py</li>
</ul>
